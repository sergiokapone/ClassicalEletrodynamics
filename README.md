# Класична електродинаміка: Конспект лекцій

Конспект лекцій з курсу класичної електродинаміки та теорії поля. Матеріал охоплює як базові рівняння Максвелла, так і релятивістську електродинаміку з використанням формалізму простору Мінковського.

## Зміст

### [Частина I](Basics/)

- [Базові поняття та рівняння](Basics/)
  - Величини, що спостерігаються в електродинаміці
  - Рівняння електромагнітного поля
  - Закони збереження
  - Межі застосовності класичної електродинаміки
- [Розв’язки рівнянь Максвелла](Solutions/)
  - Потенціали електромагнітного поля
  - Потенціали ізольованої системи зарядів і струмів
  - Задача Коші для рівнянь Максвелла
- [Вільне електромагнітне поле](FreeField/)
  - Спектральний розклад і плоскі хвилі
  - Випадкові поля випромінювання
  - Співвідношення між тривалістю сигналу та шириною його спектру
  - Загальний розв’язок рівнянь вільного поля
- [Випромінювання](Radiation/)
  - Поле заряду, що рухається з прискоренням
  - Випромінювання малих систем
  - Розсіювання електромагнітних хвиль

### [Частина II](LorentzTransform/)

- [Перетворення Лоренца як наслідок постулатів Ейнштейна](LorentzTransform/)
  - Простір-час, системи відліку
  - Перетворення Лоренца для одновимірних рухів систем відліку
  - Власний час
  - Квадрат інтервалу
  - Перетворення довжини та об’єму
- [Співвідношення СТВ у просторі Мінковського](Minkovsky/)
  - Загальні та власні перетворення Лоренца
  - Вектори і тензори в просторі Мінковського
  - Світові лінії та чотиривектор швидкості
- [Електродинаміка у просторі Мінковського](RelElecrodynamics/)
  - Чотиривимірна форма рівнянь Максвелла
  - Трансформаційні властивості тензора поля
  - Рівняння руху зарядженої частинки
  - Плоскі монохроматичні хвилі
  - Ефект Доплера
  - Вектор-потенціал електромагнітного поля у випадку ізольованої системи
- [Варіаційний принцип для рівнянь електродинаміки](VariationPrinciple/)
  - Заряд у зовнішньому електромагнітному полі
  - Дія для електромагнітного поля
- [Тензор енергії-імпульсу і закони збереження](TEI/)
  - Чотири-імпульс системи тіл
  - Енергія-імпульс електромагнітного поля
  - Тензор моменту імпульсу електромагнітної системи
